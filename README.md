# Genshin CAPTCHA
把抹茶小蛋糕，换成原神各种小蛋糕。

## 运行项目：
本项目使用Vue框架编写而成
```bash
npm install
npm run dev
```
然后根据终端提示访问 http://localhost:5173/ 即可
